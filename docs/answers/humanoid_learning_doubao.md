现在业界主流是用**模仿学习 + 强化学习 + 动作重定向 + 仿真到现实（Sim2Real）**这条流水线来教机器人学人类动作；数据来源分三类：**动捕（高质量）、遥操作（低成本）、互联网视频（大规模）**。下面从方案、关键技术、开源项目/产品三部分讲清楚。

---

## 一、主流训练方案（业界标准流水线）

### 1）数据采集：三种范式
1. **运动捕捉（Mocap）**
   - 设备：OptiTrack、Vicon、Azure Kinect、iPhone（LiDAR+视频）
   - 输出：人体 3D 关节轨迹、全局姿态、手部精细动作
   - 特点：**高质量、精准、但贵、场景受限**；工业/科研最常用
2. **遥操作（Teleoperation）**
   - 方式：人穿戴外骨骼/动捕服/手柄，直接遥控机器人做动作
   - 输出：机器人关节轨迹 + 力反馈数据
   - 特点：**低成本、直接对应机器人本体、易部署**；现在创业公司最爱用
3. **互联网视频（单目 RGB）**
   - 输入：YouTube/抖音/教学视频（无标记、无动捕）
   - 处理：用 2D→3D 姿态估计（如 VideoPose3D、HumanNeRF）重建人体运动
   - 特点：**数据无限、零成本、但噪声大、跨体型难**；学术热点、2024–2025 爆发期

### 2）动作重定向（Retargeting）：把人类动作“搬”到机器人
人 vs 机器人：骨骼长度、关节自由度、运动范围都不一样，**不能直接复制关节角度**。
- 核心方法：
  - **关节空间映射**：按比例缩放关节角度（简单、易抖）
  - **任务空间映射（末端位姿）**：保证手/足在 3D 空间轨迹一致（主流、稳定）
  - **优化/逆运动学（IK）**：加约束（关节极限、碰撞、平衡），用优化求解
- 工具：ROS（MoveIt! IK）、PyTorch IK、RoboDynamics、Unitree SDK

### 3）模仿学习（Imitation Learning, IL）：让机器人“照猫画虎”
1. **行为克隆（BC，最常用）**
   - 本质：**监督学习**，输入传感器（图像/关节），输出机器人动作
   - 模型：MLP、CNN、**Transformer（ACT/BC-T5）、Diffusion Policy**
   - 特点：**快、稳、易收敛**；但容易过拟合、泛化差
2. **逆强化学习（IRL）**
   - 思路：先从人类数据学“奖励函数”，再用 RL 学策略
   - 代表：GAIL、AIRL、AMP（适合全身运动，如行走、跳跃）
   - 特点：**泛化强、抗干扰**；但训练慢、难调参
3. **最新：视频模仿（Video Imitation，2024–2025）**
   - 代表：**VideoMimic（伯克利+宇树）、UniSkill（延世大学）、MIMICDROID**
   - 流程：单目视频 → 人体 3D 姿态 → 重定向到机器人 → 仿真训练 → 真机部署
   - 特点：**不用动捕、不用遥操作、看视频就能学**；2025 年已在宇树 G1 实现 100+ 动作

### 4）强化学习（RL）：让动作更自然、稳定、抗干扰
- 作用：**修正模仿误差、优化动态平衡、提升鲁棒性**
- 算法：PPO（最稳）、SAC、DreamerV3、APEX（2025，动作先验+多评论家）
- 奖励设计：
  - 姿态匹配（和人类示范轨迹误差）
  - 平衡奖励（ZMP、脚底压力）
  - 动作平滑、能耗惩罚
- 场景：行走、跑步、上下楼梯、跳跃、全身协调动作

### 5）仿真到现实（Sim2Real）：虚拟训练→真机直接用
- 仿真平台：**Isaac Sim（NVIDIA，业界最强）、MuJoCo、Webots、PyBullet、Genesis**
- 关键技术：
  - **域随机化（Domain Randomization）**：随机化物理参数（质量、摩擦、阻尼）、视觉纹理、光照，让策略在虚拟中见过足够多变化，真机直接迁移
  - **神经辐射场（NeRF）/HumanNeRF**：重建真实场景几何，提升仿真逼真度
- 流程：
  1. 动捕/视频 → 人体 3D 运动
  2. 重定向到仿真机器人
  3. 用 BC+RL 训练策略
  4. 直接部署到真机（**不需要再微调**，2025 主流方案）

## 二、业界典型产品/方案（2024–2025）

### 1）宇树科技（Unitree）G1/H1
- 方案：**动捕+遥操作+VideoMimic+Isaac Sim+PPO**
- 效果：G1 能模仿人类跳舞、打太极、上下楼梯、搬东西；2025 年已实现**单视频→100+动作**快速迁移


### 2）特斯拉 Optimus
- 方案：**视觉（8 摄像头）+ 自监督姿态估计 + 遥操作 BC + 仿真 RL**
- 特点：不用动捕，靠**视觉+AI**理解人类动作；2025 重点推“**技能库复用**”（先学基础动作，再组合复杂任务）

### 3）Figure AI（Figure 01）
- 方案：**全身动捕 + 遥操作 + Diffusion Policy + Sim2Real**
- 效果：开门、倒水、整理桌面等精细操作；强调**类人动作自然度**

### 4）Agility Robotics Digit
- 方案：**动捕+AMP（对抗模仿）+ 强化学习**
- 特点：双足行走、上下台阶、摔倒自恢复；动态平衡业界顶尖

## 三、开源项目（直接能用，2025 最新）

### 1）模仿学习框架
- **robomimic（ARISE-Initiative，斯坦福）**
  - 地址：https://github.com/ARISE-Initiative/robomimic
  - 功能：**行为克隆、IRL、多模态（图像/关节）、标准化数据集**；机器人操作模仿学习首选
- **LeRobot（Hugging Face，2025）**
  - 地址：https://github.com/huggingface/lerobot
  - 功能：**统一机器人学习框架、支持 BC/RL、多硬件（机械臂/人形）、云训练**；适合快速原型
- **MimicKit（NVIDIA+SFU，2025）**
  - 地址：https://github.com/xbpeng/MimicKit
  - 功能：**全身运动模仿、AMP/GAIL/PPO、支持人形机器人/虚拟角色**；适合动态动作（跑、跳、特技）

### 2）视频→动作（2025 热点）
- **VideoMimic（UC Berkeley + Unitree，2025）**
  - 地址：https://video-mimic.github.io/
  - 功能：**单目视频→人形机器人动作、无需动捕、100+动作库、支持宇树 G1**；业界首个实用化视频模仿系统
- **UniSkill（延世大学，2025）**
  - 地址：https://uniskill-robot.github.io/
  - 功能：**跨体型视频模仿、无需配对数据、87% 真实环境成功率**；泛化性强

### 3）仿真平台（Sim2Real）
- **Isaac Sim（NVIDIA，开源+商业）**
  - 地址：https://developer.nvidia.com/isaac-sim
  - 功能：**高保真物理渲染、GPU 加速、支持 Unitree/Tesla/Figure 机器人、域随机化、Sim2Real 一键部署**；业界工业级标准
- **Genesis（2025，开源）**
  - 地址：https://genesis-robotics.github.io/
  - 功能：**轻量级、快速仿真、支持人形/四足/机械臂、内置模仿学习+RL、适合学术/创业公司**

### 4）动作重定向/姿态估计
- **HumanNeRF（2024）**
  - 地址：https://humannerf.github.io/
  - 功能：**单目视频→高精度 3D 人体姿态+几何、支持动态场景、适合视频模仿输入**
- **PyTorch-IK（开源）**
  - 地址：https://github.com/facebookresearch/pytorch-ik
  - 功能：**快速逆运动学、支持任意机器人骨骼、动作重定向常用工具**
